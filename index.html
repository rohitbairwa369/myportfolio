My Projects

Pre-Interview Test Application

The Pre-Interview Test Application is a tool that allows applicants to take a test before an interview. It ensures integrity by taking random photos, capturing the applicant's location, and providing detailed reports for the admin.

Stack Used : Mean Stack

Features

  • Anti-Cheating: Random pictures of the applicant ensure they're not cheating.
  • Location Tracking: IP and location capture verify where the test is being taken.
  • Easy Test Creation: Admins can create and share tests with ease.
  • Batch Upload: Multiple applicants can be added through Excel file upload.
  • Detailed Reports: Admins get complete reports with photos, location, and test performance.
  • Scalability: Handles multiple applicants taking tests at the same time.
  • Automated Process: Reduces the need for manual supervision during the test.
  • Time-Saving: Helps filter out less suitable candidates before interviews.
  • Flexible for Applicants: Applicants can take the test at their convenience via a shared link.

During my internship at "12th Wonder," I built a Pre-Interview Test Application. I quickly learned Figma and created design mockups within 2 days. After that, I implemented the application in about a month, initially using Firebase for the backend. Later on, I transitioned the backend to Node.js.
